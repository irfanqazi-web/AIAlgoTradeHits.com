How to Get the Complete Country List

Note: Twelve Data does not publish a complete list of all 50+ supported countries on their public website. However, you can retrieve the full, current list programmatically using their API.
Method 1: Countries API Endpoint (Recommended)

Query the countries endpoint to get the complete list with ISO codes, capitals, and currencies:

GET https://api.twelvedata.com/countries

Returns:

{ "data": [ { "iso2": "US", "iso3": "USA", "numeric": "840", "name": "United States", "official_name": "United States of America", "capital": "Washington D.C.", "currency": "USD" }, ... ] }
Method 2: Exchanges API Endpoint

Get all exchanges which will show you which countries have market coverage:

GET https://api.twelvedata.com/exchanges

This returns exchange details including country names and Market Identifier Codes (MIC).

Method 3: Filter Stocks by Country

Query stocks for specific countries using 3-letter ISO codes:

GET https://api.twelvedata.com/stocks?country=USA

Replace "USA" with other country codes to explore available markets.

Python Code Example

Use this Python script to retrieve and display all supported countries:

import requests import json # Replace with your API key API_KEY = "YOUR_API_KEY_HERE" # Get all countries response = requests.get( "https://api.twelvedata.com/countries", params={"apikey": API_KEY} ) if response.status_code == 200: countries = response.json()["data"] print(f"Total Countries: {len(countries)}\n") # Print all countries for country in countries: print(f"{country['name']} ({country['iso3']}) - {country['currency']}") # Save to file with open('twelve_data_countries.json', 'w') as f: json.dump(countries, f, indent=2) print(f"\n‚úì Saved complete list to twelve_data_countries.json") else: print(f"Error: {response.status_code}") # Get all exchanges exchanges_response = requests.get( "https://api.twelvedata.com/exchanges", params={"apikey": API_KEY} ) if exchanges_response.status_code == 200: exchanges = exchanges_response.json()["data"] # Group by country by_country = {} for exchange in exchanges: country = exchange.get("country", "Unknown") if country not in by_country: by_country[country] = [] by_country[country].append(exchange["name"]) print(f"\n\nExchanges by Country:") for country, exch_list in sorted(by_country.items()): print(f"\n{country}: {len(exch_list)} exchanges") for exch in exch_list: print(f" - {exch}")
