## Business Portfolio

### 1. AIAlgoTradeHits.com (Primary Product)

**Category:** FinTech / AI Trading
**Status:** Active - Production

**Description:**
AI-powered algorithmic trading platform providing real-time market analysis, technical indicators, and trading signals for stocks and cryptocurrencies.

**Key Features:**
- Real-time stock and crypto price tracking
- 29 technical indicators (RSI, MACD, Bollinger Bands, etc.)
- AI-powered pattern recognition
- Smart search with NLP
- Voice search capabilities
- Multi-timeframe analysis (Daily, Hourly, 5-minute)
- Weekly analysis dashboards
- Portfolio tracking
- Price alerts

**Technology Stack:**
| Component | Technology |
|-----------|------------|
| Frontend | React + Vite |
| Backend API | Python Flask |
| AI/ML | Google Vertex AI (Gemini 3 Pro) |
| Database | Google BigQuery |
| Hosting | Google Cloud Run |
| Data Sources | TwelveData, Kraken Pro API |

**Current Deployment:**
| Service | URL |
|---------|-----|
| Frontend | https://trading-app-6pmz2y7ouq-uc.a.run.app |
| AI Intelligence | https://ai-trading-intelligence-6pmz2y7ouq-uc.a.run.app |
| Smart Search | https://smart-search-6pmz2y7ouq-uc.a.run.app |

**Data Assets:**
- 1.4M+ historical stock records
- 95K+ crypto records
- 527K+ ETF records
- 150K+ forex records
- Real-time data updated every 5 minutes
