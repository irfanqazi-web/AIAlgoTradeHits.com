### NAME
Data Type: STRING
Description: Full name of the trading asset or company
Trading Meaning: Human-readable name of the asset for display purposes
Example Value: Apple Inc., Bitcoin, Euro/US Dollar

### ASSET_TYPE
Data Type: STRING
Description: Classification of the asset type
Trading Meaning: Categorizes assets into: stocks, crypto, forex, etfs, indices, commodities, bonds
Example Value: Common Stock, Digital Currency, Currency Pair

### EXCHANGE
Data Type: STRING
Description: Stock exchange or trading venue where the asset is listed
Trading Meaning: Indicates market hours, liquidity, and regulatory jurisdiction
Example Value: NASDAQ, NYSE, Binance, FOREX

### COUNTRY
Data Type: STRING
Description: Country where the asset is domiciled or primarily traded
Trading Meaning: Important for tax implications, currency exposure, and geopolitical risk assessment
Example Value: United States, United Kingdom, Global

### SECTOR
Data Type: STRING
Description: Economic sector classification of the asset
Trading Meaning: Used for sector rotation strategies and portfolio diversification. Common sectors: Technology, Healthcare, Energy, Financial Services
Example Value: Technology, Healthcare, Energy, Consumer Cyclical

### INDUSTRY
Data Type: STRING
Description: Specific industry within a sector
Trading Meaning: More granular classification for industry-specific analysis and peer comparison
Example Value: Software - Infrastructure, Biotechnology, Oil & Gas

### CURRENCY
Data Type: STRING
Description: Currency in which the asset is priced
Trading Meaning: Determines currency exposure and conversion requirements for international traders
Example Value: USD, EUR, GBP, JPY

### OPEN
Data Type: FLOAT
Description: Opening price at the start of the trading period
Trading Meaning: First traded price of the period. Gap analysis uses open vs previous close. Opening range breakout strategies use this level.
Trading Signals: Gap up (open > prev_close) = bullish; Gap down (open < prev_close) = bearish
Example Value: 150.25

### HIGH
Data Type: FLOAT
Description: Highest price reached during the trading period
Trading Meaning: Intraday resistance level. Upper shadow in candlestick patterns. Used in calculating ATR, Stochastic, and range analysis.
Trading Signals: Rejection from highs = selling pressure; Breaking previous highs = bullish momentum
Example Value: 155.75

### LOW
Data Type: FLOAT
Description: Lowest price reached during the trading period
Trading Meaning: Intraday support level. Lower shadow in candlestick patterns. Used in calculating ATR, Stochastic, and range analysis.
Trading Signals: Bounce from lows = buying pressure; Breaking previous lows = bearish momentum
Example Value: 148.50

### CLOSE
Data Type: FLOAT
Description: Closing/last traded price of the period
Trading Meaning: Most important price for technical analysis. Most indicators use close price. Settlement price for many derivatives.
Trading Signals: Close near high = bullish; Close near low = bearish; Close near open = indecision (doji)
Example Value: 153.20

### PREVIOUS_CLOSE
Data Type: FLOAT
Description: Closing price from the previous trading period
Trading Meaning: Reference point for calculating daily change and identifying gaps. Used for after-hours movement analysis.
Example Value: 151.00

### VOLUME
Data Type: FLOAT
Description: Number of shares/units traded during the period

Example Value: 75000000

### AVERAGE_VOLUME
Data Type: FLOAT
Description: Average trading volume over a lookback period (typically 10-30 days)
Trading Meaning: Baseline for volume comparison. Current volume / average volume = relative volume ratio.
Trading Signals: Volume > 2x average = significant activity; Volume < 0.5x average = low interest
Example Value: 50000000

### CHANGE

Example Value: 2.20

### RSI
Data Type: FLOAT
Description: Relative Strength Index (14-period default)
Trading Meaning: Oscillator measuring speed and magnitude of price changes. Range 0-100. Most popular momentum indicator.
Trading Signals: RSI < 30 = oversold (buy signal); RSI > 70 = overbought (sell signal); Divergence with price = reversal signal
Formula: RSI = 100 - (100 / (1 + RS)) where RS = avg gain / avg loss over 14 periods
Example Value: 45.5

### STOCH_K
Data Type: FLOAT
Description: Stochastic %K (Fast Stochastic)
Trading Meaning: Shows where close is relative to high-low range. Range 0-100. More sensitive than %D.
Trading Signals: %K crossing above %D = bullish; %K crossing below %D = bearish
Formula: %K = ((Close - Lowest Low) / (Highest High - Lowest Low)) * 100
Example Value: 55.2

### STOCH_D
Data Type: FLOAT
Description: Stochastic %D (Slow Stochastic)
Trading Meaning: 3-period SMA of %K. Smoother signal line. Used for crossover signals.
Trading Signals: %K < 20 and crossing above %D = buy; %K > 80 and crossing below %D = sell
Example Value: 52.8

### MOMENTUM

### ROC

### WILLIAMS_R

### Finding Oversold Stocks (RSI < 30)
```sql
SELECT symbol, name, close, rsi, percent_change
FROM `aialgotradehits.crypto_trading_data.v2_stocks_daily`
WHERE rsi < 30 AND rsi IS NOT NULL
ORDER BY rsi ASC
LIMIT 20;
```

### Top Gainers Today
```sql
SELECT symbol, name, close, percent_change, volume
FROM `aialgotradehits.crypto_trading_data.v2_stocks_daily`
WHERE percent_change > 0
ORDER BY percent_change DESC
LIMIT 20;
```

### Stocks Breaking 52-Week High
```sql
SELECT symbol, name, close, week_52_high,
       (close / week_52_high * 100) as pct_of_high
FROM `aialgotradehits.crypto_trading_data.v2_stocks_daily`
WHERE close >= week_52_high * 0.98
ORDER BY close / week_52_high DESC
LIMIT 20;
```

### High Volume Cryptos with Bullish MACD
```sql
SELECT symbol, name, close, volume, macd, macd_signal
FROM `aialgotradehits.crypto_trading_data.v2_crypto_daily`
WHERE macd > macd_signal AND macd > 0
  AND volume > average_volume
ORDER BY volume DESC
LIMIT 20;
```

### Strong Trend Assets (ADX > 25)
```sql
SELECT symbol, name, close, adx, plus_di, minus_di,
       CASE WHEN plus_di > minus_di THEN 'Bullish' ELSE 'Bearish' END as trend_direction
FROM `aialgotradehits.crypto_trading_data.v2_stocks_daily`
WHERE adx > 25
ORDER BY adx DESC
LIMIT 30;
```

### Bollinger Band Squeeze (Low Volatility, Pending Breakout)
```sql
SELECT symbol, name, close,
       bollinger_upper - bollinger_lower as band_width,
       atr
FROM `aialgotradehits.crypto_trading_data.v2_stocks_daily`
WHERE (bollinger_upper - bollinger_lower) / bollinger_middle < 0.04
ORDER BY (bollinger_upper - bollinger_lower) / bollinger_middle ASC
LIMIT 20;
```

### Weekly Performance Analysis
```sql
SELECT symbol, name,
       close as current_price,
       percent_change as weekly_change,
       week_52_high, week_52_low,
       (close - week_52_low) / (week_52_high - week_52_low) * 100 as position_in_range
```
