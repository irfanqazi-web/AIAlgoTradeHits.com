## 6. User Flow

### Content Capture Flow
```
1. User browses Instagram/TikTok/YouTube/Any .COM website
2. User clicks browser extension OR pastes URL in web app
3. System detects content type (video/image/article)
4. Content fetched and stored in Cloud Storage
5. Metadata extracted and saved to BigQuery
6. Automatic transcription/extraction begins
7. Content added to diSearch.ai knowledge base
8. User notified when processing complete
```

### Idea Generation Flow
```
1. User selects transcribed content
2. User chooses target platform:
   - KaamyabPakistan.org (Pakistani business)
   - YouInvent.Tech (technology innovation)
   - HomeFranchise.Biz (home-based franchise)
   - NoCodeAI.Cloud (no-code AI tools)
3. System retrieves similar successful ideas from diSearch.ai
4. AI generates multiple idea variations using RAG
5. Ideas scored by relevance, feasibility, market potential
6. User reviews, edits, and selects preferred ideas
7. Ideas formatted using diWriter
8. Export to PDF, publish to platform, or save to library
```

# Extraction flow
class ContentExtractor:
    async def extract(self, url: str) -> ExtractedContent:
        # 1. Fetch page
        html = await self.fetch_with_retry(url)

        # 2. Parse HTML
        soup = BeautifulSoup(html, 'lxml')

        # 3. Extract main content (Readability algorithm)
        main_content = self.extract_main_content(soup)

        # 4. Extract metadata
        metadata = {
            "title": self.get_title(soup),
            "description": self.get_meta_description(soup),
            "author": self.get_author(soup),
            "publish_date": self.get_publish_date(soup),
            "images": self.get_images(soup),
            "videos": self.get_videos(soup),
            "links": self.get_links(soup),
            "og_data": self.get_opengraph_data(soup),
            "schema_data": self.get_schema_org_data(soup)
        }

        # 5. Clean text
        clean_text = self.clean_text(main_content)

        # 6. AI summarization (optional)
        summary = await self.ai_summarize(clean_text)

        return ExtractedContent(
            url=url,

            metadata=metadata

### Cloud Function: Web Scraper

```python
# cloud_function_scraper/main.py
import functions_framework
from playwright.async_api import async_playwright
from bs4 import BeautifulSoup
import json

@functions_framework.http
async def web_scraper(request):
    """Cloud Function for web scraping."""
    data = request.get_json()
    url = data.get('url')
    scrape_type = data.get('type', 'article')  # article, product, social

    async with async_playwright() as p:
        browser = await p.chromium.launch(headless=True)
        page = await browser.new_page()

        # Set user agent
        await page.set_extra_http_headers({
            'User-Agent': get_random_user_agent()
        })

        # Navigate and wait for content
        await page.goto(url, wait_until='networkidle')

        # Get rendered HTML
        html = await page.content()

        # Extract content based on type
        if scrape_type == 'article':
            content = extract_article(html)
        elif scrape_type == 'product':
            content = extract_product(html)
        elif scrape_type == 'social':
            content = extract_social(html)
        else:
            content = extract_generic(html)

        await browser.close()

    return json.dumps({
        'success': True,
        'url': url,
        'content': content
    })
```
